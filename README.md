# 📝 Smart Evaluate - Automated Answer Sheet Evaluation

## 📌 Overview
Smart Evaluate is a Machine Learning-based system that evaluates handwritten answer sheets automatically using image processing and text recognition techniques.

---

## 🚀 Features
- Upload handwritten answer sheets
- Image processing using OpenCV
- Text extraction using OCR
- Automated scoring system
- Fast and efficient evaluation

---

## 🛠️ Tech Stack
- Python
- OpenCV
- Tesseract OCR
- Streamlit

---

## ⚙️ How It Works
1. User uploads an answer sheet image
2. Image is processed using OpenCV
3. Text is extracted using OCR
4. System evaluates answers and generates score
