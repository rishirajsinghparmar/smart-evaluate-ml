# 📝 Smart Evaluate - Automated Answer Sheet Evaluation

## 📌 Overview
Smart Evaluate is a Machine Learning-based concept designed to evaluate handwritten answer sheets automatically using OCR (Optical Character Recognition) and Computer Vision.

---

## 🚀 Features
- Automated answer sheet evaluation
- Handwritten text recognition (HTR)
- Image preprocessing using OpenCV
- Scoring system based on extracted answers
- Reduces manual evaluation effort

---

## 🛠️ Tech Stack
- Python
- OpenCV
- OCR (Tesseract)
- Machine Learning

---

## ⚙️ How It Works
1. Answer sheet is scanned/uploaded as an image
2. Image preprocessing is applied (noise removal, thresholding)
3. OCR extracts handwritten text
4. Extracted answers are compared with model answers
5. System generates a score automatically
